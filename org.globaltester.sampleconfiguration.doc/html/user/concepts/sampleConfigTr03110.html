TR-03110

The tab TR03110 allows to provide details on supported features related to BSI TR-03110.

The section Supported features allows to select the support for the following features:
CSTA EF.CardSecurity is protected by TA
CardInfo EF.CardAccess or EF.CardSecurity contains a CardInfo element.
PSAInfo EF.CardAccess or EF.CardSecurity contains a PSAInfo element.
PrivilegedTerminalInfo EF.CardAccess or EF.CardSecurity contains a PrivilegedTerminalInfo element.
