Auxillary Data Verification

The tab AuxData allows to provide details needed for Auxillary data verification.

The section Protocol support allows to select whether or not the sample supports auxillary data verification at all. If you don't check the box AuxData all other options on this tab are irrelevant. The checkbox CMP must be checked if the sample supports verification using the COMPARE command.

The section Preferred parameters allows to provide needed input for the verification tests. If auxillary data verification is supported the field Community ID shall contain the hex encoded complete value of the valid Community ID stored in the sample. This will be used (with different modifications) from within the testcases as valid value to be checked against.
