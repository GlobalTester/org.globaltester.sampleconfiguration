Active Authentication

The tab AA allows to provide details about ActiveAuthentication.

The section Protocol support allows to select whether or not the sample supports ActiveAuthentication at all. If you don't check the box AA all other options on this tab are irrelevant.

The section Supported features allows to select which variants of ActiveAuthentication are supported by the sample. Select ECDSA if your sample uses elliptic curves for ActiveAuthentication or RSA for RSA based ActiveAuthentication.

The section Required options is needed for mechanism specific configurations. The switch Use extended length for AA needs to be checked iff the personalisation of your sample enforces APDUs that exceed the length of short APDUs, i.e. when you use large RSA keys.
