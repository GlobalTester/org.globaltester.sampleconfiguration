Passwords

The tab Passwords allows to provide passwords to the card, required to execute tests, as well as some features related to password management.

The section Supported passwords allows to provide passwords to be used during test. All the values shall be represented in their default user readable form, i.e. no conversion to HexString shall be performed before entering the values here.

The section Supported passwords allows to select which password management functions are supported according to the following descriptions:
CNG_PIN_PUK sample allows to change the PIN after PUK authentication
CNG_PIN_AR sample allows to change the PIN after terminal authentication with appropriate access rights
CNG_CAN_AR sample allows to change the CAN after terminal authentication with appropriate access rights
