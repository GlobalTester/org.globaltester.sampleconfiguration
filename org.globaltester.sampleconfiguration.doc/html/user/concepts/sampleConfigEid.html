eID Application

The tab eID allows to provide details on supported features of the eID application.

The section Protocol support allows to select whether or not the sample contains an eID application at all. If you don't check the box eID all other options on this tab are irrelevant.

The section Supported data groups allows to select which datagroups are present within the eID application on the sample.

The field CS in section Supported EF_X must be selected iff your sample contains an EF_ChipSecurity.
