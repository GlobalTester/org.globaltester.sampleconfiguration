TestData

The folder TestData is automatically created in each TestSpecification. Data used for the TestCases can be stored here.
TestData can be card-verifiable certificates stored in the XML structure for certificates or sample configurations.

This way all test data is stored with the TestSpecification.
With the XML structure, card-verifiable certificates will be generated for the execution of the TestCases within a TestSpecification, or they can be transformed in a human-readable form if the TestSpecification is exported to the Open Document Format.
Subroutines will be evaluated in the Java Script code, and now can be exported to the Open Document Format as well.
