Terminal authentication v2

The tab TAv2 allows to provide details needed for Terminal Authentication v2.

The section Protocol support allows to select whether or not the sample supports terminal authentication version 2 at all. If you don't check the box TA all other options on this tab are irrelevant.

The section Supported features describes which variants of TA are supported according to the following descriptions:
RSA terminal authentication based ond RSA
ECDSA terminal authentication based on elliptic curve cryptography
MIG migration of the cryptosystem for TA
DATE validation of certificate dates
AUTH_EXT authorization extensions are interpreted
