ICAO

The tab ICAO allows to provide details on supported features related to ICAO Doc9303.

The section Protocol support allows to select the support for the following features:
ICAO checking this box implies the sample is an ICAO complaint passport (i.e. contains the minimum needed datagroups etc.)
Plain checking this box implies the sample is an ICAO plain passport, that does support selection of the ePassport application w/o secure messaging

The section Supported features contains configuration options to test for specific characteristics of a given sample. The dropdown LDS version provides information on the version of the LDS specification implemented in the sample.
